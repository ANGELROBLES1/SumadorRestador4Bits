# Sumador y Restador de 4 Bits

Este proyecto implementa un sumador y restador de 4 bits utilizando únicamente compuertas lógicas AND, OR y NOT.

## Contenido
- Implementación en Python del sumador-restador
- Pruebas de funcionamiento
- Informe en PDF
- Simulación del circuito (Proteus)

- Python
- Proteus
- Lógica digital
